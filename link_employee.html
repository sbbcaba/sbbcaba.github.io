Finalizing the Link

Linking is a two-save process. One save for the search popup, one save for the profile.

1st Save: Search Popup

Clicking save in the search popup maps the ID but does not yet commit it. You are halfway done.

2nd Save: Main Profile

CRITICAL: Once the popup closes, you MUST click the "SAVE" button on the main employee profile page to finalize the link permanently.
